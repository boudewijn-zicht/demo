# datocms.schema.json

To handle 'possibleTypes' Apollo needs to have a schema.  This schema can be generated by datocms using the graphql query:

```
query Schema {
  __schema {
	    types {
        name
        kind
        possibleTypes {
          name
        }
      }
  }
}
```
